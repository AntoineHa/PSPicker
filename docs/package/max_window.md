  Function made to get the maximum in a user defined window   
     
  Input:   
      first_sample: first sample of the window   
      last_sample: last sample of the window   
      data: input array   
         
  Output:   
      MAX: maximum value   
      IND: maximum index   
         
  Example:   
      [MAX,IND]=max_window(first_sample,last_sample,trace,flag_plot)
